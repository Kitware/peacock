=============
peacock-trame
=============

MOOSE GUI developed using Trame from Kitware Inc.


* Free software: GNU General Public License v3 (GPLv3)


Installing
----------
Build and install the Vue components

.. code-block:: console

    cd vue-components
    npm i
    npm run build
    cd -

Install the application

.. code-block:: console

    pip install -e .


Run the application

.. code-block:: console

    peacock-trame
